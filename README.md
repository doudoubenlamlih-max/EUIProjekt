# Marktplatz & Bidding - EUI Projekt

## Projektbeschreibung
Eine Webanwendung für einen Online-Marktplatz mit Bidding-System.
Gebaut mit Laravel (MVC Architektur).

## Technologien
- PHP / Laravel 11
- MySQL
- HTML / CSS
- JavaScript (Fetch API)

## Team
- douaebenlamlih

## Installation
1. Repository klonen
2. `composer install`
3. `.env` Datei konfigurieren
4. `php artisan migrate`
5. `php artisan serve`

## Funktionen
- Benutzer Registrierung & Login
- Produkte erstellen / bearbeiten / löschen
- Bieten auf Produkte
- Kategorien filtern & sortieren
- Admin Bereich
